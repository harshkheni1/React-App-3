### Box Component

You can wrap your other components into Box component for styling them

### Spacing
```
we can make static and responsive spacings examples:
static:
<Box mt={20} pb={30}>test</Box>
responsive:
<Box mt={{ xs: 20, md: 30 }} pb={{ xs: 20, md: 30 }}>test</Box>
```

<Preview>
    <Box bg="primary" p={[10, 20, 30, 40]} m={[10, 20, 30, 40]}>Margin and padding static</Box>
    <Box
      bg="primary"
      p={{ xs: [10, 20, 30, 40], sm: [12, 22, 32, 42], md: [14, 24, 34, 44], lg: [16, 26, 36, 46], xl: [18, 28, 38, 48] }}
      m={{ xs: [10, 20, 30, 40], sm: [12, 22, 32, 42], md: [14, 24, 34, 44], lg: [16, 26, 36, 46], xl: [18, 28, 38, 48] }}
    >
      Margin and padding responsive
    </Box>
</Preview>
