### Basic InputGroup

Modal consists of three sections Header, Body and Footer. All of these you can directly use from Modal component only like bellow . . .

```typescript
  <InputGroup>
    <InputGroup.Block>
      <FormInput name="search" type="text" />
    </InputGroup.Block>
    <InputGroup.Addon>
      <Button color="primary">
        <Icon name="search" measure={18} />
      </Button>
    </InputGroup.Addon>
  </InputGroup>
  <InputGroup>
    <InputGroup.Addon>$</InputGroup.Addon>
    <InputGroup.Block>
      <FormInput name="test" type="text" />
    </InputGroup.Block>
    <InputGroup.Addon>%</InputGroup.Addon>
  </InputGroup>
```
