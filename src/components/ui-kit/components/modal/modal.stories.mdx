# Modal

### Basic Modal

Modal consists of three sections Header, Body and Footer. All of these you can directly use from Modal component only like bellow . . .

```typescript
<Modal isOpen onClose={changeModalState(false)}>
    <Modal.Header>Add User</Modal.Header>
    <Modal.Body>
        <Input id="name-input" type="text" fullWidth label="Name" placeholder="Enter your Name" />
    </Modal.Body>
    <Modal.Footer>
        <Button variant="outlined">Add User</Button>
    </Modal.Footer>
</Modal>
```
